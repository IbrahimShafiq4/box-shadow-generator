# Box-Shadow Generator Project

Welcome to the Box-Shadow Generator project! This project provides a simple tool for generating CSS box-shadow effects using HTML, CSS, and JavaScript. Customize and preview box-shadow styles effortlessly with this intuitive interface.

## Introduction
The Box-Shadow Generator project aims to simplify the process of creating CSS box-shadow effects for web developers and designers. With this tool, users can dynamically adjust parameters such as shadow color, blur radius, spread radius, and shadow offset to achieve desired visual effects.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/IbrahimShafiq4/box-shadow-generator.git
   ```
2. Navigate to the project directory:
   ```bash
   cd box-shadow-generator
   ```
3. Open `index.html` in your web browser to launch the Box-Shadow Generator.

## Usage
1. Use the sliders or input fields to adjust the box-shadow parameters.
2. View the real-time preview of the box-shadow effect on the sample element.
3. Copy the generated CSS code to use in your own projects.
